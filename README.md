# 英语闪卡 | English Flashcards

一个基于 SM-2 间隔重复算法的英语单词学习 Web App。

## 功能

- 📚 40个精选英语词汇（含音标、例句、中文翻译）
- 🔄 3D卡片翻转动画
- 🧠 SM-2间隔重复算法（再来/困难/掌握/简单）
- 📊 学习进度追踪（localStorage持久化）
- 🔍 实时搜索过滤
- 📱 触屏滑动支持
- ⌨️ 键盘快捷键
- 🌙 暗色模式自适应

## 使用方法

直接用浏览器打开 `index.html`，或使用任意静态文件服务器：

```bash
npx serve .
# 或
python3 -m http.server 3000
```

## 键盘快捷键

| 按键 | 功能 |
|------|------|
| 空格 / F | 翻转卡片 |
| ← / P | 上一张 |
| → / N | 下一张 |
| 1 | 再来（重置） |
| 2 | 困难 |
| 3 | 掌握 |
| 4 | 简单（加速间隔） |
